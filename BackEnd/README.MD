# **Retruck-Back-End**  
Developed in ASP.NET Core 6.0, the back-end implements the business rules of Retruck.  
It is connected to a SQL Server database. :computer:  

# **Services Used**  
### FireBase :blue_book:  
### Google Maps (Distance Matrix) :postbox:  
### Holidays API :date:  
### Position Stack API :outbox_tray:
